---
### **1. Install JDK**
- Download and install the latest **Java Development Kit (JDK)** from the  
  [Oracle JDK](https://www.oracle.com/java/technologies/javase-jdk21-downloads.html)  
  or [OpenJDK](https://openjdk.org/install/) website.
- Set the **JAVA_HOME** environment variable to point to the JDK installation directory.
  - Add the JDK's `bin` directory to your system's **PATH** variable.   
  - This allows you to run Java commands from the command line.
---
### **2. Install IntelliJ IDEA**
- Download and install **IntelliJ IDEA** from the  
  [JetBrains website](https://www.jetbrains.com/idea/download/).
- You can choose either the **Community Edition** (free) or the **Ultimate Edition** (paid).
---
### **3. Create a New Java Project in IntelliJ**
1. Open IntelliJ IDEA.
2. Click on **"New Project"**.
3. Select **"Java"** from the list of project types.
4. Choose the **JDK** you installed earlier.
5. Click **"Next"** and then **"Finish"** to create the project.
---
### **4. Write Your First Java Program**
1. In the **Project** view, right-click on the `src` folder.
2. Select **New > Java Class**.
3. Name the class `HelloWorld`.java
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
---
### **5. Run the Java Program**
1. Right-click on the `HelloWorld.java` file in the Project view.
2. Select **Run 'HelloWorld.main()'**.
3. The output **"Hello, World!"** should appear in the Run window at the bottom of IntelliJ.
