## **Variables & Constants**

--- 
### **1. Variables**
**Variables** are **containers that store data in memory**.  
A variable is used to **initialize and store data** that can change during  
program execution.

---

## **Key Components of Variables in Java**
A variable in Java has **three components**:

- **Data Type** – Defines the type of data stored  
  *(e.g., `int`, `float`, `double`, `String`)*
- **Variable Name** – A unique identifier that follows Java naming rules
- **Value** – The actual data assigned to the variable

**Example:**
```java
int age = 25;
