## **Keywords & Identifiers (Java)**

---
### **Keywords**
**Keywords** are **reserved words** in Java that have a **predefined meaning**  
to the compiler. They **cannot be used** as variable names, method names,  
class names, or identifiers.

Java keywords are used to **define syntax**, **control flow**, **data types**,  
and **access rules**.

**Examples of Java Keywords:**
- **class**
- **public**
- **static**
- **void**
- **int**
- **double**
- **if**
- **else**
- **for**
- **while**
- **switch**
- **return**
- **try**
- **catch**
- **finally**
- **new**
- **this**
- **super**
- **extends**
- **implements**
- **interface**
- **package**
- **import**
- **throws**
- **throw**
- **break**
- **continue**

✅ **Important Rules for Keywords**
- Keywords are **case-sensitive**
- They are **fixed** by the Java language
- They **cannot be redefined**

---

### **Identifiers**
**Identifiers** are **user-defined names** given to:
- Packages
- Classes
- Interfaces
- Methods
- Variables
- Objects

They are used to **identify program elements**.

**Examples of Identifiers:**
- `Student`
- `main`
- `age`
- `calculateTotal`
- `employeeSalary`

---

### **Rules for Naming Identifiers**
1. Must start with a **letter (A–Z or a–z)**, **underscore (_) **, or **dollar sign ($)**
2. Cannot start with a **digit**
3. Can contain **letters, digits, underscores, and dollar signs**
4. Must not be a **Java keyword**
5. Cannot contain **spaces or special characters**
6. Identifiers are **case-sensitive**

✅ Valid Identifiers:
- `count`
- `_total`
- `$price`
- `employeeAge`
- `MyClass`

❌ Invalid Identifiers:
- `2value` ❌ (starts with digit)
- `class` ❌ (keyword)
- `total-amount` ❌ (contains hyphen)
- `first name` ❌ (contains space)

---

### **Difference Between Keywords and Identifiers**

| **Keywords** | **Identifiers** |
|-------------|----------------|
| Reserved words | User-defined names |
| Have predefined meaning | Represent program elements |
| Cannot be renamed | Can be freely named |
| Part of Java syntax | Used by programmers |
